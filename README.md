🎨 Jannah Loraine Ada | UI/UX Portfolio
A modern, responsive portfolio built with Next.js and TypeScript, featuring a fully functional contact form and dynamic dark mode.

🚀 Live Demo
(https://ada-portfolio-next.vercel.app/)

✨ Features
Dynamic Theme Switching: Seamless toggle between light and dark modes.

Contact System: Automated email delivery using Nodemailer and Gmail API.

Responsive Design: Fully optimized for mobile, tablet, and desktop views.

Smooth Animations: CSS-based transitions for a polished user experience.

🛠️ Tech Stack
Framework: Next.js (App Router).

Styling: Tailwind CSS & Custom CSS Variables.

Backend: Next.js API Routes.

Email: Nodemailer.

📦 Installation & Setup

Clone the repo:
Bash
git clone https://github.com/jannahnah/ada-portfolio-next.git

Install dependencies:
Bash
npm install
Configure Environment Variables: Create a .env.local file and add your Gmail credentials:

Code snippet
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-password

Run development server:
Bash
npm run dev
